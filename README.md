# PowMr Home Assistant Add-on

Це локальний MQTT-проксі для інверторів PowMr (моделі RWB1 та аналогічні).

## Як встановити
1. Додайте цей репозиторій у Home Assistant (Settings -> Add-ons -> Add-on Store -> Repositories).
2. Встановіть "PowMr Inverter Bridge".
3. Налаштуйте перенаправлення трафіку на вашому роутері (DNS або Static Route для IP хмари PowMr на IP вашого HA).
4. Запустіть додаток.

## Функції
- Автоматичне виявлення (Auto-Discovery) всіх сенсорів.
- Перехоплення даних у реальному часі.
- Робота без зовнішніх хмар (якщо налаштовано локально).
